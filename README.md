# Detonato

A stopwatch/counter and pomodoro timer built with OpenGL, using [Posy's beautiful 7-segment digit designs](https://www.youtube.com/watch?v=RTB5XhjbgZA).

## Building

Requires GLFW3 and a C compiler.

```sh
make
./detonato
```
